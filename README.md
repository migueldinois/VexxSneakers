# 🛒 Vexx Sneakers - E-commerce

Este é um projeto de uma loja virtual (E-commerce) completa, focada na venda de tênis e calçados. O projeto abrange desde a navegação de produtos por categorias até o fluxo de autenticação e comentarios de usuarios

---

## 📐 Sobre o Design (Wireframe)

O design foi planejado para ser minimalista, moderno e focado na experiência do usuário (UX), contando com as seguintes telas mapeadas:

1. **Home / Catálogo Geral:** Banner principal, blocos de categorias em destaque e listagem geral de produtos.
2. **Filtro por Categoria:** Visualização de produtos de uma categoria única selecionada.
3. **Detalhes do Produto:** Página dedicada com imagem ampliada, preço, botão de adicionar ao carrinho e seção de comentários dos usuários.
4. **Carrinho de Compras:** Lateral ou página dedicada para conferência de itens selecionados e valor total.
5. **Autenticação (Cadastro & Login):** Telas com formulários limpos para criação de conta e acesso ao sistema.
6. **Rodapé (Footer):** Links institucionais (Políticas de Privacidade, Termos de Uso) e redes sociais.
